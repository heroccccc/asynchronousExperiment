# asynchronousExperiment

### キーボードでビデオの切り替えを(自然に?)行うシステム
* #### 実験で使用
* #### 異なるビデオに切り替わる際にブラックアウト等なしで、自然に切り替わる（ビデオが切り替わっているのかわからないくらい）  

---
### 主に使用したライブラリ
* ####  opencv
* ####  PyQt
* ####  PyAudio
---
### 使用する前に
* ####  動画（ビデオ）を複数用意
* ####  その動画の音声を別に用意
  * ####  [ffmpeg](http://ffmpeg.org)等でビデオ映像と音声を分割できます
---

* ####  ビデオ切り替わる際にブラックアウト等なしで切り替わっています
